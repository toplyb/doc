# toRef 函数

  • toRef 是一个函数, 可以将响应式对象 (复杂类型数据) 中某一个属性单独转换成响应式对象, 并且值是关联的

  • 经过 toRef 转换的响应式对象, 其中 属性.value 中存的是该属性的值, 修改的时候需要是 属性.value = 新值

    <div>
    {{name}}
</div>
<button @click="updateName">
    改变
</button>
<script>
import { reactive, toRef } from "vue";
export default {
  setup() {
    const obj = reactive({
      name: "xiaoming",
      age: 12,
    });
    // 如果模板中只需要使用 name 属性,则可以这样写, 但是此时 name 不是响应式数据
    // 从响应式对象数据中结构出来的属性数据, 不是响应式数据
    // const {name} = obj

    // 这样写, 可以将其变成响应式数据
    const value = toRef(obj, "name");

    const updateName = () => {
      name.value = "xiaohong";
    };

    return { name, updateName };
  },
};
</script>
