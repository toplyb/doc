# v-cloak

  • 作用: 当模板还没有被解析的时候, 不会显示双大括号的内容

  • 用法如下:

    <style>
[v-cloak] {
  display: none;
}
</style>
<template id="ceshi">
  <h2 v-cloak>hello world</h2>
</template>

<script>
Vue.createApp({
  template: "#ceshi",
}).mount("#app");
</script>

# v-bind

  • 作用: 用来绑定属性

  • 用法如下:

    <template id="ceshi">
  <img v-bind:src="img" />
  <a v-bind:href="link">百度一下</a>
</template>

// 语法糖
<template id="ceshi">
  <img :src="img" />
  <a :href="link">百度一下</a>
</template>

<script>
Vue.createApp({
  template: "#ceshi",
  data() {
    return {
      img: "1.png",
      link: "http://www.baidu.com",
    };
  },
}).mount("#app");
</script>

    # 绑定类名

    <template id="ceshi">
  // 动态绑定字符串
  <div :class="className">123</div>

  // 动态绑定对象
  <div :class="{ active: status, title: true }">123</div>

  // 动态绑定数组
  <div
    :class="[
      'zxc',
      className1,
      isActive ? 'active' : '',
      { active: isActive1 },
    ]"
  >
    123
  </div>

  // 默认 class 和动态 class 结合
  <div class="abc" :class="{ active: status, title: true }">123</div>
</template>

<script>
Vue.createApp({
  template: "#ceshi",
  data() {
    return {
      className: "ceshi",
      status: true,
      className1: "title",
      isActive: true,
      isActive1: true,
    };
  },
}).mount("#app");
</script>

    # 动态绑定属性

    <template id="ceshi">
  <div :[acb]="value">123</div>
</template>

<script>
Vue.createApp({
  template: "#ceshi",
  data() {
    return {
      abc: "ceshi",
      value: "1",
    };
  },
}).mount("#app");
</script>

    # 绑定对象中的所有属性

    <template id="ceshi">
  <div v-bind="info">123</div>
  // 这样绑定最后的结果是
  <div name="xiaoming" age="12"></div>
</template>

<script>
Vue.createApp({
  template: "#ceshi",
  data() {
    return {
      info: {
        name: "xiaoming",
        age: 12,
      },
    };
  },
}).mount("#app");
</script>

# v-click

  • 点击事件

  • 当点击事件不传递参数的时候, 如果想要获取到鼠标的点击对象, 则函数中的第一个参数就表示点击对象, 调用函数的时候不需要传递参数

  • 当点击事件传递参数的时候, 如果想要获取到鼠标的点击对象, 那么调用函数的时候需要在第一个位置传递 $event 参数, 其他的参数则写在后面

    <template id="ceshi">
<p>{{ name }}</p>
<div @click="change">123</div>
</template>

<script>
Vue.createApp({
    template: '#ceshi',
    data() {
        return {
            name: 'xiaoming'
        }
    },
    methods: {
        change() {
            ...
        }
    }
}).mount('#app')
</script>

# v-show 和 v-if 的区别

  • v-show 不支持 template
  • v-show 不能和 v-else 一起使用
  • v-show 是通过 CSS 中的 display 来隐藏的, v-if 是直接不渲染标签
  • 如果需要需要在显示和隐藏之间频繁的切换, 则使用 v-show
  • 如果没有频繁的切换, 则使用 v-if
