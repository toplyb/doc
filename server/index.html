# 字符串

# mb_substr

  • 返回字符串中的一部分内容, 其中, substr()只针对英文字符, 如果是汉字的话, 则需要使用 mb_substr()

    <?php
     # (string) mb_substr(string $str, int $start [, int $length = null, string $encoding = mb_internal_encoding()])
     echo mb_substr('xiaosheng', 1) # iaosheng
     echo mb_substr('xiaosheng', 1, 2) # ia

# str_pad

  • 使用指定字符串将另外一个字符串 (从左, 两边, 右) 填充到指定长度

    <?php
    # (string) str_pad(string $input, int $length, [string $string, $type]);
    # $string 表示要以什么字符串填充
    # 其中 $type 有三种类型, STR_PAD_RIGHT, STR_PAD_LEFT, STR_PAD_BOTH, 如果没有指定的话, 默认是 STR_PAD_RIGHT
    # 分别表示从左, 两边, 右填充
    $input = 'xiaosheng';
	echo str_pad($input,3);  # 'xiaosheng'
	echo str_pad($input,33);  # 'xiaosheng'
	echo str_pad($input,13,'_');  # 'xiaosheng____'
	echo str_pad($input,12,'_',STR_PAD_BOTH);  # '_xiaosheng__'
	echo str_pad($input,13,'_',STR_PAD_BOTH);  # '__xiaosheng__'

# array_flip

  • 交换数组中的键值 (可以用来去除数组中重复的值)

    <?php
    $arr1 = ['xiaoming','xiaosheng','xiaolan'];
	$arr2 = array_flip($arr1);
	print_r($arr2);
	# 输出结果为 :
	# ['xiaoming' => 0,'xiaosheng' => 1,'xiaolan' => 2]
	#---------------------------------------------------------------------
	$arr1 = ['name' => 'xiaoming','age' => 23,'old' => 23];
	$arr2 = array_flip($arr1);
	print_r($arr2);
	# 输出结果为 :
	# ['xiaoming' => 'name',23 => 'old']

# array_walk

  • 给二维数组中所有的数组添加同样的字段
<?php
      $arr = ['age' => 11];
      $list = array(
          ['id'=>1,'name'=>'aaa'],
          ['id'=>2,'name'=>'bbb'],
          ['id'=>3,'name'=>'ccc']
      );
      array_walk($list, function (&$value, $key, $arr) {
          $value = array_merge($value, $arr);
      },$arr);
      var_dump($list);
  /*['id'=>1,'name'=>'aaa','age' => 11],['id'=>2,'name'=>'bbb','age' => 11],['id'=>3,'name'=>'ccc','age' => 11]*/

# 三元运算符

<?php
      $a = 0;
      if ($a) {
          $b = $a;
      } else {
          $b = '2';
      }	# 以上可以简写为: 	$b = $a ? $a : '2';	$b = $a ? : '2';	# 最后结果都为2

# ??

<?php
      $a = 0;
	  $b = $a ?? '1'; # 等同于 isset($a) ? $a : 0;	# 结果为0

# & 引用

<?php
      $a = 10;
	  $b = &$a;	# 此时, $a 和 $b 是指向了同一个内容, 他俩是完全相同的

# 常用命令

# echo()

  • eccho 是一种语言结构, 并不是真正的函数, 没有返回值
  • 可以输出多个参数
  • 如果输出内容中包含有变量, 需要用双引号来解析变量, 否则输出的是变量名, 而不是变量值
  • 输出字符串, 数字

# print()

  • print() 是一种语言结构, 不用用括号包围参数
  • print() 只支持一个参数
  • 如果输出内容中包含有变量, 需要用引号来解析变量, 否则输出的是变量名, 而不是变量值

# printf()

  • printf() 是一个函数, 返回一个格式化后的字符串

# print_r()

  • 是一个函数, 可以输出字符串, 数字, 数组
  • 输出布尔值和 null 没有意义, 因为都是打印 \n

# vam_dump()

  • 输出变量的长度, 类型和内容

  • 打印的信息比 print_r() 的详细

# include

  • include() 每次执行的时候都会加载一次文件
  • 在用到的时候才加载
  • 如果遇到错误的话, 会给出提示并继续执行下面的代码

# require

  • require() 只有第一次执行的时候才会加载一次

  • 一开始就加载

  • 如果遇到错误的话, 直接抛出致命错误, 下面的代码都不会执行

  • include_once() 和 require_once 跟上面两个的用法差不多, 只不过是只加载一次

# 函数

# isset

  • 判断变量是否定义, 返回 false 的有以下几种情况:

    • 变量定义了但是没有赋值
    • 变量值为 null

  • 其他情况都返回 true

# empty

  • 判断是否为空, 返回 true 的有以下几种情况:
    • 变量定义了但是没有赋值
    • 变量值为 null
    • 变量值为空字符串
    • 变量值为 false
    • 变量值为 0
    • 变量值为 '0'

# is_null

  • 判断是否为 null, 返回 true 的有以下几种情况:
    • 变量定义了但是没有赋值
    • 变量值为 null

# 魔术方法

  • __construct(): 构造函数
  • __destruct(): 析构函数
  • __call($fun, $args): 在对象中调用一个未定义或者不可访问的方法时被调用
  • __callStatic($fun, $args): 当调用一个未定义或者是不可访问的静态方法的时候被调用
  • __get($propertyName): 获取一个类成员变量时被调用
  • __set($property, $value): 设置一个类成员变量时被调用
  • __isset($content): 当调用 isset 或者 empty 对一个未定义或者不可访问的成员赋值时被调用
  • __unset($content): 当调用 reset 对一个未定义或者不可访问的成员更新时被调用
  • __sleep(): 执行 serialize 时, 会先调用这个函数
  • __wakeup(): 执行 unserialize 时, 会先调用这个函数
  • __toString(): 当使用 echo 方法直接输出对象时被调用
  • __invoke(): 当使用调用函数访问一个对象时该方法会首先被调用
  • __set_state($an_array): 调用 var_export 导出类时被调用
  • __clone(): 当对象被复制赋值时该方法被调用
  • __autoload($className): 尝试加载一个未定义的类时被调用
  • __debuginfo: 输出 debug 信息时被调用

# OOP

  • self 是对当前类的静态引用, 取决于定义当前方法所在的类. 即 self 写在哪个类, 它就指的是谁. 可以用于访问类的静态属性, 静态方法和常量
  • $this 指的是实际调用的对象, 哪个对象调用, 他就指的是谁. 但是 $this 不能访问类中的静态属性和常量, 且不能存在于静态方法中
  • static 除了可以声明类的静态成员 (属性和方法) 外, 还可以用于后期静态绑定. 可以用于静态或者非静态方法中, 也可以访问类的静态属性, 静态方法, 常量和非静态方法, 但是不能访问非静态属性. 静态调用时, static 指向的是实际调用的类, 非静态调用时, static 指向的是实际调用时对象所属的类
