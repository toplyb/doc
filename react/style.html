# 使用sass

  • 由于在使用 create-react-app 创建项目的时候, 内置已经配置好了 sass 环境, 所以只需要使用 npm i sass -D 安装一下即可
  • 如果想要使用 less, 则需要自己配置

# css模块化

  • 当使用 import ./index.css 的方式导入 css 的时候, 如果整个 html 中有相同的类名, 则会引起样式覆盖冲突的问题, 所以这时可以使用以下方式导入
// css 文件一般以 .module.css 结尾
import classes from './index.module.css'

const Index = () => {
    return (
        // box 是类名, index 是模块名, 最终显示在浏览器上的并不是 box, react 会对这个类名重新命名, 这样就防止了类名重复引起的问题
    	<div className={classes.box}></div>
    )
}

export default Index
