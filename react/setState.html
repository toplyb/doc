# setState简介

  • setState 是异步更新数据的, 而且可以在一个方法中多次调用 setState, 但是, 当多次调用的时候, 不要依赖上一次 set 的结果

  • 当在一个方法中多次调用 setState 的时候, 只会执行一次 render 方法

  • 当在一个方法中多次调用 setState, 并且想要在当前拿到上一次 setState 后的值的话, 可以使用以下写法:

this.setState((state, props) => {
    // 此时 state 就表示的是上一次 setState 后的值, 也就是最新的
    
    // 但是, 此时 setState 仍然是异步的
})
  • setState 第二个参数是接收一个回调函数, 该回调函数会在状态更新后立即执行, 也就是说如果想要立马拿到 setState 更新后的值并且做些事情的话, 就可以在该回调函数中书写, 如:
class App extends React.Component {
  state = {
    count: 0
  }

  increment = () => {
    this.setState((state, props) => {
      return {
        count: state.count + 1
      }
    }, () => {
      console.log(this.state.count); // 1
    })

    console.log(this.state.count) // 0
  }

  render() {
    return (
      <div>
        <p>{this.state.count}</p>
        <button onClick={this.increment}>+1</button>
      </div>
    )
  }
}

ReactDOM.render(<App></App>, document.getElementById('root'))
